Bootstrap Basics

What is Bootstrap?

Bootstrap is a free, open-source CSS framework that provides pre-built components and utilities for creating responsive, mobile-first websites quickly and easily.

  • Responsive grid system
  • Pre-built components
  • Utility classes
  • Cross-browser compatibility
<!-- Bootstrap CDN -->
<link href="bootstrap.min.css" rel="stylesheet">
<!-- Basic Bootstrap layout -->
<div class="container">
<div class="row">
<div class="col-md-6">
Content here
</div>
</div>
</div>

Grid System

Use Bootstrap's responsive 12-column grid system to create flexible layouts that adapt to different screen sizes automatically.

Components

Access a library of pre-built components like navigation bars, cards, modals, forms, and buttons that are ready to use.

Responsive Design

Build mobile-first websites with responsive breakpoints and utilities that work seamlessly across all devices.

Customizable

Customize Bootstrap with Sass variables, custom CSS, and theme options to match your brand and design preferences.

Best Practices

Do's
  • Use Bootstrap's grid system properly
  • Customize with CSS variables
  • Follow mobile-first approach
  • Use semantic HTML with Bootstrap classes
  • Optimize for performance
Don'ts
  • Don't override Bootstrap styles unnecessarily
  • Avoid using too many utility classes
  • Don't forget to test on different browsers
  • Avoid mixing Bootstrap versions
  • Don't ignore accessibility guidelines
